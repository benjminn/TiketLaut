## â“ FAQ (Frequently Asked Questions)

### Umum

**Q: Apakah TiketLaut tersedia di platform lain?**  
A: Saat ini hanya tersedia untuk Windows Desktop. Versi mobile dan web sedang dalam pengembangan.

**Q: Apakah data saya aman?**  
A: Ya, kami menggunakan enkripsi SSL dan database secure di Supabase. Password di-hash dengan bcrypt.

**Q: Bisakah saya booking tanpa akun?**  
A: Tidak, Anda harus registrasi terlebih dahulu untuk tracking dan notifikasi.

### Booking & Pembayaran

**Q: Berapa lama proses verifikasi pembayaran?**  
A: Maksimal 1x24 jam (biasanya 1-3 jam di hari kerja).

**Q: Bagaimana jika upload bukti transfer yang salah?**  
A: Anda bisa upload ulang setelah ditolak admin, atau contact customer service.

**Q: Bisakah saya membatalkan booking?**  
A: Ya, dengan syarat:
- H-3 atau lebih: Refund 50%
- H-2 sampai H-1: No refund
- H-0: No refund

**Q: Metode pembayaran apa yang paling cepat?**  
A: E-Wallet (GoPay/OVO/DANA) karena realtime, tapi tetap perlu upload screenshot.

**Q: Bagaimana jika jadwal dibatalkan oleh perusahaan?**  
A: Refund 100% otomatis atau reschedule gratis ke jadwal lain.

### Tiket

**Q: Apakah e-ticket sama dengan tiket fisik?**  
A: Ya, e-ticket memiliki kekuatan hukum sama. Tunjukkan e-ticket (print/digital) saat boarding.

**Q: Bagaimana jika kehilangan e-ticket?**  
A: Login ke akun â†’ Riwayat â†’ Download ulang. Atau gunakan kode booking di Cek Booking.

**Q: Bisakah transfer tiket ke orang lain?**  
A: Tidak, tiket tied to nama penumpang di booking. Jika perlu ubah, contact customer service.

**Q: Apakah tiket bisa digunakan untuk jadwal lain?**  
A: Tidak, tiket hanya valid untuk jadwal yang dibooking. Perlu rebooking untuk jadwal berbeda.

### Kendaraan

**Q: Apakah semua jenis kendaraan bisa dibawa?**  
A: Tergantung kapasitas kapal. Umumnya sepeda motor, mobil sedan/SUV, dan truk kecil bisa.

**Q: Bagaimana cara menghitung biaya kendaraan?**  
A: Sistem otomatis hitung berdasarkan grup kendaraan yang dipilih.

**Q: Apakah driver kendaraan perlu tiket terpisah?**  
A: Ya, driver harus booking tiket penumpang juga (include dalam total penumpang).

### Teknis

**Q: Aplikasi tidak bisa connect ke server?**  
A: Cek koneksi internet. Jika masih error, cek status Supabase di status.supabase.com

**Q: Lupa password, tapi email tidak masuk?**  
A: Cek folder Spam/Junk. Jika tidak ada, contact customer service dengan proof of identity.

**Q: Aplikasi crash saat startup?**  
A: Update .NET Runtime atau reinstall aplikasi. Cek troubleshooting di Installation Guide.

**Q: Data pengguna tidak tersimpan?**  
A: Pastikan connection string di appsettings.json benar dan database accessible.

### Admin

**Q: Bagaimana cara tambah admin baru?**  
A: Insert manual ke table Admin di database atau gunakan fitur User Management (role upgrade).

**Q: Bisakah admin membuat booking untuk customer?**  
A: Belum tersedia di versi ini. Customer harus booking sendiri.

**Q: Bagaimana handle dispute payment?**  
A: Review bukti pembayaran dengan teliti. Jika ragu, contact user untuk konfirmasi.

**Q: Data analytics tidak update?**  
A: Klik tombol Refresh di dashboard. Analytics update setiap 5 menit otomatis.
